Navbar examples taken directly from getbootstrap.com/docs/4.4/examples/navbars/

Based on Navicon Transformicons by Bennett Feely.

How to Use

Add one of these classes (x uarr larr rarr plus minus) to the navbar-toggler div.
Replace the default icon with three spans using the icon-bar class as shown in the examples below.
